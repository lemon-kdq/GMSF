IMU 支路

  • VQF — Vector Quaternion Filter，IMU 6D 融合输出四元数姿态
  • 航位推算 — 轮速计提供前进速度，IMU 提供姿态方向，积分得到位置
  • 杆臂补偿 — 补偿 IMU 与车体旋转中心之间的偏距
  • 输出 — 全频轨迹初值，作为 GTSAM 里程计约束

LiDAR 支路

  • 关键帧选择 — 位移 >10m / 偏航角 >15° / 横滚俯仰 >1° / 时间间隔 >10s 触发
  • 点云正畸 — 扫描过程中车辆运动，按时间分 batch 补偿到目标时刻
  • ICP 匹配 — 当前帧点到前一帧平面特征的点到面距离作为优化约束
  • 输出 — 矫正点云 + 关键帧位姿，作为 GTSAM 空间约束

GTSAM 优化

  • 因子图 — 节点 = 关键帧位姿，因子 = Prior / Between / Attitude / Point-to-Plane
  • 里程计因子 — 来自航位推算，约束相邻关键帧相对运动
  • 点到面因子 — 点云 ICP 匹配残差，Huber 鲁棒核处理离群点
  • 求解 — Levenberg-Marquardt，非线性最小二乘，最多 20 次迭代
